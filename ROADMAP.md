# Roadmap

## Short-term (v1.1 — v1.3)

- [ ] Sound effects toggle (mute/unmute) in HUD
- [ ] Mobile responsiveness — canvas scaling, touch controls
- [ ] Keyboard shortcut hints overlay (press `?`)
- [ ] Confirmation dialog before leaving an active game
- [ ] Challenge difficulty indicators (★☆☆ / ★★★)
- [ ] Game timer / elapsed time display
- [ ] Hover animation on StoneButton components
- [ ] Keyboard navigation for Level Select
- [ ] Friendly error messages in code challenges
- [ ] Loading skeleton for Level Select

## Mid-term (v1.4 — v2.0)

- [ ] More levels — community-sourced puzzle pack
- [ ] Level editor — let players create and share custom puzzles
- [ ] Multiplayer races — solve challenges against other players
- [ ] Enhanced leaderboard — seasonal resets, tier rankings
- [ ] Achievement system — badges for milestones
- [ ] Dark mode toggle (beyond current theme)
- [ ] Localization / i18n support
- [ ] PWA improvements — offline play for completed levels

## Long-term (v2.0+)

- [ ] Plugin API — third-party challenge integrations
- [ ] Advanced level editor with visual scene builder
- [ ] Procedural level generation
- [ ] Corun Creator program — featured community challenges
- [ ] Mobile native app (React Native)
- [ ] Full save-state sync across devices via Supabase

---

*This roadmap is a living document. Priorities may shift based on community feedback.*
