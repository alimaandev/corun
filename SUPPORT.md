# Support

## Issues

Report bugs or request features via [GitHub Issues](https://github.com/alimaandev/corun/issues).

Before opening a new issue, please:
- Search existing issues to avoid duplicates
- Use the issue template (bug report / feature request)
- Include screenshots or console output when relevant

## Discussions

Ask questions, share ideas, or show what you built in [GitHub Discussions](https://github.com/alimaandev/corun/discussions).

## Security

Report security vulnerabilities to the repository's security advisory or contact the maintainer directly.
